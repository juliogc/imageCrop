$.imageCrop()

About the options

allowMove

Specifies if the selection can be moved (default value is true).

allowResize

Specifies if the selection can be resized (default value is true).

allowSelect

Specifies if the user can make a new selection (default value is true).

aspectRatio

Specifies the grow proportion of the selection (default value is 0).

imagePath

URL path from the background image from the selection (default is "scripts/libs/imageCrop/outline.gif").

minSelect

The minimum area size to register a new selection (default value is [0, 0]).

minSize

The minimum size to make a new selection (default value is [0, 0]).

maxSize

The maximum size to make a new selection (default value is [0, 0]).

msPolyfill

Path to the polyfill file that will fix issues in IE8 from the 'background-size' CSS3 property (default is "scripts/libs/imageCrop/background-size-polyfill.htc").

outlineOpacity

The outline opacity (default value is 0.5).

overlayBgColor

The overlay background color (default is "#fff").

overlayOpacity

The overlay opacity (default value is 0.5).

selectionPosition

The selection position (default value is [0, 0]).

selectionWidth

The selection width (default value is 0).

selectionHeight

The selection height (default value is 0).
